--- layout: main title: LODQA ---
LODQA (Linked Open Data Question Answering) is an open source project aiming at developing a system to generate SPARQL queries from natural language queries.
  • (Oct. 19/20, 2014) NLIWoD 2014 Workshop will be collocated with ISWC2014 in Trentino.
  • (Feb. 17-21, 2014) OKBQA 2014 Hackathon is held in Jeju.
  • (Oct. 01, 2013) Relation detection is improved to find the right pairs of terms in relation.
  • (Dec. 20, 2013) LODQA is launched as an open source project.
